# 侯夫人與殺豬刀 - Chinese Novel Reader

A web-based Chinese novel reader with chapter navigation and translation support.

## Features

- 📖 **172 Chapters** - Complete novel with proper chapter separation
- 🔍 **Chapter Navigation** - Dropdown menu and prev/next buttons
- ⌨️ **Keyboard Shortcuts** - Arrow keys (← →) for quick navigation
- 🌐 **Translation Ready** - Optimized for Chrome's built-in translator
- 💾 **Auto-Save Progress** - Remembers your last reading position
- 📱 **Responsive Design** - Works on desktop and mobile devices

## How to Use

1. Open `侯夫人與殺豬刀_chapters.html` in Google Chrome
2. Right-click anywhere on the page
3. Select "Translate to Thai" (or your preferred language)
4. Navigate using:
   - Chapter dropdown selector
   - Previous/Next buttons
   - Arrow keys (← →)

## Files

- `侯夫人與殺豬刀.txt` - Original Chinese text file
- `侯夫人與殺豬刀_chapters.html` - **Main reader** (with chapter navigation)
- `侯夫人與殺豬刀.html` - Simple HTML version (no chapter breaks)
- `convert_improved.sh` - Script to regenerate the chapter-based HTML
- `convert_to_chapters.sh` - Alternative conversion script
- `convert_to_html.sh` - Script for simple HTML conversion

## Technical Details

- Pure HTML/CSS/JavaScript - no dependencies
- Fixed navigation bar for easy access
- LocalStorage for progress tracking
- Clean, readable layout optimized for long-form reading

## Browser Requirements

- **Recommended**: Google Chrome (for translation feature)
- Any modern browser with JavaScript enabled
