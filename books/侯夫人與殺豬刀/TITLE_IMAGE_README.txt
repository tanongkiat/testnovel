📖 Book Cover Image Instructions
================================

To add a custom cover image for this book:

1. Place your cover image in this directory with the filename: title.jpg

2. Recommended specifications:
   - File format: JPEG (.jpg)
   - Aspect ratio: 2:3 (portrait, like a book cover)
   - Recommended size: 600x900 pixels or larger
   - File size: Keep under 500KB for faster loading

3. After adding the title.jpg file, run the generator script:
   python generate_book_index.py

4. The index page will automatically display your cover image

Note: If title.jpg doesn't exist, a placeholder book icon (📖) will be shown instead.
